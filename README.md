# Приложение с виджетом обратного отсчета

Android-приложение с виджетом для обратного отсчета времени.

## Возможности

- Виджет на главном экране с обратным отсчетом
- Настройка времени (часы, минуты, секунды) в приложении
- Автоматическое обновление виджета каждую секунду

## Как использовать

1. Откройте проект в Android Studio
2. Соберите и установите приложение на устройство
3. Откройте приложение и выберите время обратного отсчета
4. Нажмите "Запустить таймер"
5. Добавьте виджет на главный экран (долгое нажатие → Виджеты → Обратный отсчет)

## Требования

- Android Studio Arctic Fox или новее
- Android SDK 24 (Android 7.0) или выше
- Gradle 8.2.0
